Quick health checks

Create order:

curl -X POST http://localhost:8080/orders \
-H "Content-Type: application/json" \
-d '{"customer_id":"101","restaurant_id":"22"}'

Get order:

curl "http://localhost:8080/get/order?order_id=1"

docker exec -it food-delivery-postgres psql -U admin -d food_delivery

curl -X POST http://localhost:8080/assign/driver \
-H "Content-Type: application/json" \
-d '{
  "order_id":"1",
  "driver_id":"50"
}'

Check Kafka logs:

docker logs food-delivery-kafka

For just the latest lines:

docker logs --tail 20 food-delivery-kafka
